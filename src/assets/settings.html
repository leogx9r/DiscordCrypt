DiscordCrypt Settings



Primary Cipher:

This indicates the cipher used to encrypt the inner ciphertext of a message.

Secondary Cipher:

This indicates the cipher used to encrypt the outer ciphertext of a message.

Cipher Padding Mode:

This indicates the message padding scheme used for both ciphers.

Cipher Block Operation Mode:

This indicates the block mode of operation for both ciphers.

Exchange Algorithm:

This indicates the default algorithm used for handshakes.

Default Encryption Password:

This is the default password used to decrypt messages in channels without a password being set.

Decrypted Message Prefix:

This indicates what will be added to the start of a decrypted message. Can be empty.

Message Trigger:

The suffix at the end of a typed message to indicate whether to encrypt the text.

Example: This message will be encrypted.|ENC

Timed Message Expiration:

This indicates how long after an encrypted message is sent, should it be deleted in minutes.

Set this to "0" to disable timed messages.

New Master Database Password:

If specified, your master password will be updated to this.
